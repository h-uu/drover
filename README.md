# drover
To unblock Discord from your country

# Important note:
Before activating Drover, you must close Discord from Task Manager for it to function correctly. Then open Drover, select "Direct," and click "Install." If you see the message "Installation complete!", the installation is complete. Click "OK," and you're done. Now, try restarting Discord. If you see an error message saying "Please exit Discord before continuing," the installation failed because Discord is still running. Make sure to close it completely.
